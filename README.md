## 工单系统在哪里，怎么进

登录 DMIT 后台（Client Area）之后，顶部导航栏找「Support」，下拉里有「Open New Ticket」和「My Tickets」两个入口。

- **Open New Ticket**：新建工单
- **My Tickets**：查看历史工单和当前进度

第一次用的人容易找不到，因为 DMIT 的后台界面是标准 WHMCS 框架，布局比较朴素，没有什么引导提示。直接记住路径：Support → Open New Ticket，就行了。

提交工单需要选择「Department」，常见的几个分类：

- **Sales**：购买前咨询、套餐疑问
- **Technical Support**：服务器故障、网络问题、IP 相关
- **Billing**：账单、付款、退款
- **Abuse**：滥用举报（一般用不到）

选错部门会被转单，响应时间会拉长——这是我踩过的坑，第一次把技术问题发到了 Billing，多等了将近一天。

## 实际响应速度是什么水平

说实话，DMIT 的工单响应不算快，但也不是那种石沉大海的类型。

我自己的体验：

- 技术类工单，工作时间内一般 **4-8 小时**内有人回复
- 账单类问题，有时候当天回，有时候第二天
- 换 IP 这种操作性请求，回复之后执行很快，基本当天处理完

有一点要注意——DMIT 团队规模不大，时区上偏向北美，如果你在北京时间白天提单，可能正好是他们那边的深夜，等到第二天早上才回是正常的。别在提完工单两小时后就开始催，没用，也没必要。

我换 IP 那次，提单说明了原因（IP 被 GFW 封锁，附上了检测截图），大概 6 小时后回复确认，当天晚些时候 IP 就换好了。整个过程没有什么废话，直接处理——这点我觉得挺好的。

## 提工单的正确姿势（少走弯路）

工单写得好不好，直接影响处理速度。我见过有人工单就写"我的服务器坏了"，然后等客服来回问三四次才搞清楚问题，白浪费两天。

**几个实用建议：**

1. **写清楚服务器信息**：VPS 的 hostname 或者 IP 地址，在 Client Area 里可以找到，直接复制粘贴进去
2. **描述问题的时间线**：什么时候开始出问题，之前有没有做过什么操作
3. **附上截图或日志**：ping 不通就截 ping 结果，SSH 报错就贴错误信息，越具体越好
4. **说清楚你想要的结果**：是要换 IP、要退款、还是要技术排查，直接说

工单语言用英文。DMIT 是美国公司，客服团队处理英文工单更顺畅，中文也能看懂，但回复速度和质量上英文更稳。

## 换 IP、退款、账单问题——这几类工单怎么写

### 换 IP

DMIT 的换 IP 政策：部分套餐支持付费换 IP，价格一般是 $4-5 一次，具体以工单回复为准。

工单模板参考：


Subject: Request for IP Replacement - [你的服务器 IP]

Hi,

My server IP [xxx.xxx.xxx.xxx] has been blocked and is no longer accessible from mainland China.
I would like to request an IP replacement.

Server: [Hostname 或 Service ID]
Current IP: [当前 IP]

Please let me know the fee and process. Thank you.


简洁、信息完整，处理起来最快。

### 退款

DMIT 官方有退款政策，新购服务在一定时间内可以申请退款（具体条款以官网为准）。退款工单发到 Billing 部门，说明订单号和退款原因。

### 账单疑问

如果发现被多扣费或者有不认识的账单项，同样发 Billing，附上账单截图和订单号，不要只说"我的账单有问题"——说清楚哪一笔、金额是多少、你认为应该是多少。

## 常见问题

### DMIT 工单支持中文吗？

可以用中文提交，客服能看懂。但回复通常是英文，处理速度上英文工单更顺畅一些。如果英文不熟练，中文也没问题，不影响处理结果。

### 工单提交后多久没回复算异常？

正常情况下 24 小时内会有初次回复。超过 48 小时没有任何回应，可以在原工单下追加一条消息催一下，不要重复开新工单——重复开单反而会打乱队列。

### DMIT 可以换机房吗？

不支持直接迁移。如果想换机房，通常的做法是退掉当前套餐（在退款窗口内）、重新购买目标机房的套餐。超出退款期的话，就只能等到续费周期结束再换。

### IP 被墙一定能换吗？

不一定。DMIT 部分套餐支持换 IP，但不是所有套餐都有这个选项，也不保证换到的新 IP 一定干净。提工单之前可以先问一下当前套餐是否支持换 IP。

### DMIT 有实时客服或在线聊天吗？

没有。DMIT 目前只有工单系统，没有在线聊天窗口。所有支持请求都通过工单处理。

### 工单系统登录不进去怎么办？

先确认邮箱和密码是否正确，DMIT 后台和服务器的登录是分开的。如果忘记密码，用注册邮箱走「Forgot Password」流程重置。实在登不进去，可以发邮件到 support@dmit.io 说明情况。

### 购买前的咨询也可以提工单吗？

可以，发到 Sales 部门。比如问某个机房的路由情况、库存什么时候补货、套餐能不能升配，这些都可以提前问清楚再下单。
